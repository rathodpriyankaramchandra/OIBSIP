# LearnHub - Online Learning Platform

## Project Overview
LearnHub is a simple and responsive landing page for an online learning platform. It is designed to help students explore online courses and develop industry-ready skills.

## Features
- Attractive Landing Page
- Responsive Design
- Hero Section
- Popular Courses Section
- Smooth Scrolling Navigation
- Modern UI Design

## Technologies Used

- CSS3
